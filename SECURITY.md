# Security Policy

## Supported Versions

This project is intended for research, educational, and defensive security purposes.

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |

## Reporting a Vulnerability

If you discover a security issue in this project, please open a private security advisory on GitHub or contact the maintainer before public disclosure.

Please include:

- A description of the issue
- Steps to reproduce
- Expected behavior
- Actual behavior
- Platform information
- Proof of concept (if applicable)

## Responsible Disclosure

Please allow a reasonable amount of time for investigation and remediation before publicly disclosing vulnerabilities.

## Scope

This project is designed as a **passive Wi-Fi monitoring and intrusion detection system**.

It does **not** implement packet injection, denial-of-service, deauthentication attacks, or any offensive wireless capabilities.

The software is intended solely for defensive security research and educational use.
